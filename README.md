# Analisis Sentimen Topik Googlenews

Analisis Sentimen untuk Topik Berita di Gogglenews sederhana dalam Bahasa Indonesia

Analisis Sentimen adalah teknik NLP (Natural Language Processing/Pemrosesan Bahasa Alami) yang umum digunakan untuk menentukan apakah teks/kalimat itu positif, negatif, atau netral. Sering digunakan untuk melihat Kepuasan Pelanggan berdasarkan Sentimen Ulasan atau berfungsi sebagai perspektif tambahan ketika melakukan analisis pada data teks.

Pada modul python ini berisi kode Analisis Sentimen terhadap Topik Berita Online Berbahasa Indonesia tanpa harus menterjemahkan ke dalam Bahasa Inggris terlebih dahulu. Dan untuk Membantu merancang dan membuat skor sentimen berdasarkan logika Anda sendiri.
